# Update-Desktop

## SYNOPSIS
Refresh the Windows Explorer Shell, which causes the desktop icons and the environment variables to be reloaded.

## SYNTAX

```
Update-Desktop [[-ContinueOnError] <Boolean>] [<CommonParameters>]
```

## DESCRIPTION
Refresh the Windows Explorer Shell, which causes the desktop icons and the environment variables to be reloaded.

## EXAMPLES

### EXAMPLE 1
```
Update-Desktop
```
